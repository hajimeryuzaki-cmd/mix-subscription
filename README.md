# Canaria Records — 歌ってみたMIX サブスク

## ファイル構成（5ファイル）

- `index.html` — トップページ（サブスクプラン）
- `single.html` — 単発依頼ページ
- `colors_and_type.css` — 色・余白・フォントの定義
- `styles.css` — ベース読み込み
- `components.css` — 全コンポーネントのスタイル

## 編集の仕方

すべてHTMLとCSSだけで書かれているので、テキストエディタで直接編集できます。

- 文言・料金の変更：`index.html` または `single.html` を直接編集
- 色やフォントの変更：`colors_and_type.css` の変数値を変更
- レイアウトの調整：`components.css` を編集
